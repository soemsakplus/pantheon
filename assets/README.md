# Pantheon Brand Assets

## Files

- `pantheon-logo.svg` - horizontal logo for README headers, decks, and documentation.
- `pantheon-mark.svg` - square mark for icons, avatars, favicons, and social previews.

## Concept

The mark combines the project's two central metaphors:

- **Temple:** Pantheon as the place where a growing set of specialist agents lives.
- **Kernel / OS:** the central node path maps `root -> main -> multi-agents -> sub-agents`.

The four columns represent the four canonical tiers in the architecture. The gold roof reads as the Pantheon/temple layer, the teal side nodes are persistent multi-agents, and the red lower node is the task-scoped sub-agent layer.
