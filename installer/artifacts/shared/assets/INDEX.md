# Assets index

> Catalog of binary assets in `shared/assets/`. main appends rows when root drops a new file and asks to index ("main, index รูปนี้").

## Conventions

- **Path** is relative to `shared/assets/` (e.g., `people/alice.jpg`, not `shared/assets/people/alice.jpg`)
- **Subfolders** by category, created on demand: `people/`, `logos/`, `diagrams/`, `screenshots/`, `presentations/`, …
- **Filenames** kebab-case and descriptive: `alice-headshot.jpg`, not `IMG_0421.jpg`
- main does NOT generate / write binary files (POLICY §2.1). root drops the file manually; main only updates this index.
