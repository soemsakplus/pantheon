## Conventions

- **Name** = full name as you'd write in a formal email.
- **Preferred address** = how to refer to them in conversation (nickname, first name, etc.).
- **Contact** = primary channel (email or chat handle); not a place to store secrets.
- **Timezone** = IANA name preferred (e.g., `Asia/Bangkok`).
- For PII like home address, phone — do NOT store here. (Future: `shared/private/` tier when PII handling is added.)
