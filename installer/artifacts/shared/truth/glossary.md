## Conventions

- **Term** = exactly as root types it (case-sensitive — `OKR` vs `okr`).
- **Expansion** = what it stands for (for acronyms) or the definition (for codenames).
- **Context** = where this term shows up (e.g., "engineering planning", "marketing weekly").
- One row per term. If the same string means different things in different contexts, add separate rows distinguished by Context.
